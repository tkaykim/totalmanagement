# ASTCOMPANY ERP 레이아웃 설계안

## 1. 개요

ReactStudio 대시보드를 참고하여 ASTCOMPANY ERP의 통일된 레이아웃을 설계합니다.
광고대행 업무에 특화된 기능들을 포함합니다.

## 2. 페이지 구조

### 2.1 라우팅
- 경로: `/astcompany` 또는 `/ast`
- 컴포넌트: `src/app/astcompany/page.tsx`
- 대시보드 컴포넌트: `src/features/astcompany/components/ASTCompanyDashboard.tsx`

### 2.2 메인 레이아웃
ReactStudio와 동일한 사이드바 + 메인 컨텐츠 영역 구조

```
┌─────────────────────────────────────────────────┐
│  Header (로고, 사용자 정보, 알림)                  │
├──────────┬──────────────────────────────────────┤
│          │                                      │
│ Sidebar  │  Main Content Area                  │
│ (메뉴)   │  (탭별 컨텐츠)                        │
│          │                                      │
│          │                                      │
└──────────┴──────────────────────────────────────┘
```

## 3. 메뉴 구조 (사이드바)

### 3.1 메뉴 항목
ReactStudio와 유사하되 광고대행 업무에 맞게 조정:

1. **대시보드** (LayoutDashboard)
   - 전체 현황 요약
   - 프로젝트별 매출/지출/순수익
   - 진행 중인 프로젝트
   - 다가오는 일정

2. **프로젝트 관리** (Briefcase)
   - 프로젝트 목록
   - 프로젝트별 상세 정보
   - 프로젝트별 매출/지출/순수익
   - 프로젝트별 할일 관리

3. **채널 관리** (Youtube)
   - 관리 중인 유튜브 채널 목록
   - 채널별 상세 정보 (채널명, 제작사, 구독자, 다음 업로드일, 광고현황)
   - 채널별 콘텐츠 일정

4. **크리에이터 관리** (Users)
   - 크리에이터/셀럽 목록
   - 크리에이터별 상세 정보
   - 크리에이터별 프로젝트 이력
   - 크리에이터별 계약 정보

5. **외주인력 관리** (UserCheck)
   - 외주인력 목록
   - 외주인력별 상세 정보
   - 외주인력별 프로젝트 이력
   - 외주인력 정산 관리

6. **거래처 관리** (Building2)
   - 클라이언트 목록
   - 클라이언트별 상세 정보
   - 클라이언트별 프로젝트 이력

7. **정산/회계** (Receipt)
   - 매출/지출 내역
   - 프로젝트별 재무 현황
   - 기간별 집계
   - 정산 상태 관리

8. **할일 관리** (CheckSquare)
   - 전체 할일 목록
   - 프로젝트별 할일
   - 상태별 필터링 (todo, in-progress, done)

9. **일정/캘린더** (Calendar)
   - 전체 일정 보기
   - 프로젝트별 일정
   - 이벤트 타입별 필터링

10. **매뉴얼/가이드** (BookOpen)
    - 매뉴얼 목록
    - 카테고리별 분류
    - 검색 기능

## 7. API 엔드포인트

```
/api/creators
  GET: 크리에이터 목록 조회
  POST: 크리에이터 생성
/api/creators/[id]
  GET: 크리에이터 상세 조회
  PUT: 크리에이터 수정
  DELETE: 크리에이터 삭제

/api/freelancers
  GET: 외주인력 목록 조회
  POST: 외주인력 생성
/api/freelancers/[id]
  GET: 외주인력 상세 조회
  PUT: 외주인력 수정
  DELETE: 외주인력 삭제

/api/project-creators
  GET: 프로젝트-크리에이터 연결 조회
  POST: 프로젝트-크리에이터 연결 생성
/api/project-creators/[id]
  PUT: 프로젝트-크리에이터 연결 수정
  DELETE: 프로젝트-크리에이터 연결 삭제

/api/project-freelancers
  GET: 프로젝트-외주인력 연결 조회
  POST: 프로젝트-외주인력 연결 생성
/api/project-freelancers/[id]
  PUT: 프로젝트-외주인력 연결 수정
  DELETE: 프로젝트-외주인력 연결 삭제
```

## 8. 데이터 필터링

모든 데이터는 `bu_code = 'AST'`로 필터링하여 AST BU 데이터만 표시합니다.

## 9. 스타일 가이드

- ReactStudio와 동일한 디자인 시스템 사용
- Tailwind CSS 유틸리티 클래스 활용
- shadcn-ui 컴포넌트 활용
- 반응형 디자인 (모바일/태블릿/데스크톱)

## 10. 주요 기능 요구사항

### 10.1 프로젝트별 매출/지출/순수익
- 프로젝트 상세 뷰에서 실시간 계산
- 대시보드에서 프로젝트별 요약 표시
- 정산/회계 탭에서 전체 재무 현황 표시

### 10.2 크리에이터와 채널 분리
- 크리에이터 관리 탭: 개인 크리에이터/셀럽 관리
- 채널 관리 탭: 유튜브 채널 관리
- 크리에이터와 채널은 선택적으로 연결 가능

### 10.3 외주인력 정산
- 프로젝트별 외주인력 할당
- 작업량(시간/일수) 기록
- 자동 정산 금액 계산
- 정산 상태 관리

### 10.4 프로젝트 할일 관리
- 프로젝트별 할일 생성/수정/삭제
- 상태별 필터링 (todo, in-progress, done)
- 우선순위 설정
